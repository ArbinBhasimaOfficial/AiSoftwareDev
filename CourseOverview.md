    The Roadmap
      Four phases. Twelve weeks. One career.
        Each phase builds on the one before it. No phase assumes knowledge that was not already covered, and no week is filter.
          Week 1 to 3
            Core Engineering Foundations
              -> How the web works: browser, server, DNS, and HTTP
              -> JS runtime, closure, prototypes, and the event loop
              -> TS fundamentals and generics
              -> Git branching, pull requests, and npm or pnpm workflows
              -> Data structures, problem solving, and testing with vitest
              -> Clean code with proper error handling and logging
              Key Project
                Command Line task manager built in TS with no frameworks

          Week 4 to 6
            Frontend and Backend APIs
              -> HTML, CSS, and Tailwind for responsive interfaces
              -> React and Next.js component architecture
              -> Hooks, state management, and data fetching
              -> Node.js and express service-layer architecture
              -> PostgreSQL and Prisma fundamentals
              -> REST API design with an introduction to GraphQL
              Key Project
                A deployed Next.js app and a database backed API

          Week 7 to 9
            Database, Authentication and Infrastructure
              -> Advanced SQL: CTEs, window functions, and transactions
              -> Indexing and query optimization with Explain Analyze
              -> Production database patterns, integration testing, and jobs
              -> Redis, sessions, JWT, and role based access control
              -> OAuth, security middleware, and real time with webSockets
              -> Docker and Docker Compose
              -> Deployment with Vercel, Render, CloudFlare, DNS, SSL and Github Actions.
              Key Project
                A secured, real time API running in a Dockerized production stack

          Week 10 to 12
            AI Engineering, Capstone and Job Readiness
              -> AI integration with OpenAI, OpenRouter, and the Vercel AI SDK
              -> Streaming responses and structured outputs
              -> RAG, embeddings, and pgvector
              -> LangChain, LangGraph basics, and tool calling
              -> Payments with stripe, esewa, and khalti
              -> Job Readiness: resume, linkedin, github, and mock interviews
              -> Remote work readiness, freelancing, and Demo Day
              Key Project
                A production-grade full stack AI application, presented at Demo Day
