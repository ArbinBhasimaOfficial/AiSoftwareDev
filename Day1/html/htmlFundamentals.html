What is HTML?

HyperText Markup Language.

HTML describes the structure and meaning of web content -- headings, paragraphs, links, images, forms and sections.

Important: HTML is a markup language, not a programming language

It labels content so browsers know what each part is, not how to compute anything.

HTML elements

Open, content, close. That's it.

HTML attributes

Extra information attached to an element.
